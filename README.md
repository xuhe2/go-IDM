# build

```shell
make build
```

# run

use `-h` arg to get help docs

```shell
./bin/go-IDM -h
```

- `-t` to use threads
- `-n` to set file name
- `-p` to set path to save file
- `-f` to skip check file transfer method

## example

```shell
./bin/go-IDM https://d1.music.126.net/dmusic/NeteaseCloudMusic_Music_official_2.10.13.202675_32.exe
```
